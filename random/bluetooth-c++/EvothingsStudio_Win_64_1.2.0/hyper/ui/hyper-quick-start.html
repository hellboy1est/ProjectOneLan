Quick-Start Guide

Getting started instantly

Click the Run button! This opens a project in the desktop browser.

Connect from a mobile browser

On your mobile, enter the connect URL displayed in the bottom panel below the project list. Then click Run!

Note that mobile devices must be on the same WiFi network as this computer to be able connect!

Adding a new project

Drag and drop an HTML file to the project list to add it to the list!

Automatic reload

Editing and saving a file in the running project will automatically reload it!

Use the HyperReload app

Hybrid app development is easy with the HyperReload app. Get the app on the Apple AppStore (iOS) or on Google Play (Android).

The HyperReload apps gives you:

  • A full-screen browser window that feels like a native app.
  • Access to native device functionality like camera, sensors, and advanced networking.
  • Ability to easily share HTML5/JS apps with customers and friends.
  • A way to sponsor the HyperReload project.

When development is done, deploy your finished app without modifications using Apache Cordova.
