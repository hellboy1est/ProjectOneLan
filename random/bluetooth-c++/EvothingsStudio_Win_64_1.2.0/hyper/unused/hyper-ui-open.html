Quick Start

HyperOpen is a tool to let you quickly reload applications when saving changes to files in the currently running project.

Enter the full path and name of the main HTML file of your project, then click the Run button to open the file in the desktop browser.

Connect from a mobile web browser using the connect URL displayed in the Server Info panel. Then click Run. Mobile devices must be on the same WiFi network as this computer to be able connect!

Editing and saving a file in the running project will automatically reload it!
