Enter the name of your mbed compatible BLE board and tap Connect. When connected you can toggle LED1 on the mbed enabled BLE board.

This app is paired with the custom GATT mbed code. The mbed code demonstrates how to create a custom GATT service on the microcontroller side. This JavaScript application demonstrates how to interact with that custom GATT service from the phone side.
